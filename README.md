# High-Performance Rubik's Cube Engine & 3D Visualizer

A full-stack Rubik's Cube solving pipeline featuring a highly optimized C++ mathematical engine and a custom Python/OpenGL 3D interactive frontend.

## 🚀 Project Overview
This project was built to explore advanced Data Structures and Algorithms (DSA). It combines low-level bitwise optimizations with advanced graph-theory search algorithms to solve complex Rubik's Cube permutations in milliseconds. 

The backend communicates asynchronously with a custom-built 3D PyQt5 GUI, allowing users to watch the algorithm's optimal solutions executed in real-time.

## ✨ Key Features
* **C++ Bitboard Architecture:** Represents the 43 quintillion possible cube states using 64-bit integers instead of arrays, allowing for zero-overhead physical rotations and microsecond state transitions.
* **Pattern Databases (PDBs):** Replaced standard heuristics with an 84MB Corner Pattern Database. The database was generated by running a massive Breadth-First Search (BFS) algorithm to map over 88 million unique corner permutations.
* **IDA* Search Algorithm:** Utilizes Iterative Deepening A* alongside the PDB to aggressively prune the search tree, breaking the exponential time-complexity barriers of brute-force solvers.
* **Python/OpenGL GUI:** A responsive desktop application built with PyQt5 and PyOpenGL. Features interactive 3D rendering, WCA-compliant random scramble generation, and precise visual syncing.
* **Inter-Process Communication (IPC):** Seamless text-based pipeline routing inputs from the Python UI to the compiled C++ engine, parsing the mathematical output back into visual 3D animations.
